About Me

Hello! I'm Nick. I like to build things.

I am a Software Engineering student at McMaster University (Canada), just about done my degree and hoping to start a career as a Game Designer. I am a tech enthusiast and would describe myself as an entry level:

  • Software Engineer
  • Game Developer
  • Web Designer
  • Illustrator
  • Circus Performer

With the time I have that isn't dedicated to school I enjoy messing around with Photoshop, Illustrator, and any kind of code I can get my hands on. I am a huge fan of the SVG file format and get a great deal of satisfaction from squashing a bug in code.
